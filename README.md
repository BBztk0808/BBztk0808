## Featured Project

### GeoBottle (In Progress)

> Proximity-based messaging platform: drop virtual bottles at real GPS locations and discover hidden local stories.

- **Status**: Not open source yet
- **Architecture**:
  - Mobile: React Native + React 18
  - Backend A: Node.js + Express + MongoDB
  - Backend B (production line): Python FastAPI + PostgreSQL + Redis
- **Engineering Highlights**:
  - Monorepo + multi-platform + dual-backend design
  - Dockerized deployment + CI testing + OpenAPI contract tests
  - Built for fast iteration and scalability
